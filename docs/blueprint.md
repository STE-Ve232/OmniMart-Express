# **App Name**: OmniMart Express

## Core Features:

- Extensive Product Catalog: Browse and search a diverse range of products from various categories like fresh produce, electronics, apparel, and office supplies. Users can filter by category, price, and availability.
- AI-Powered Product Detail Tool: Utilize a generative AI tool to enhance product descriptions, tailor them for different customer segments (e.g., highlighting health benefits for families, durability for students), and ensure all necessary details (like usage instructions for electronics) are clearly presented.
- Dynamic Shopping Cart: Add various product types to a single cart, with the system intelligently flagging items that require special delivery considerations (e.g., chilled for perishables vs. standard for durables).
- User Authentication & Profiles: Secure user accounts allowing registration, login, and management of personal information, shipping addresses, and preferences, including a wishlist feature.
- Streamlined Checkout Process: An intuitive checkout flow that consolidates mixed-item orders, presents dynamic delivery/pickup options based on product types, and integrates placeholder payment methods suitable for Kenya.
- Order Tracking & History: Customers can view their past orders, track current order status, and review delivery details directly from their user profile.

## Style Guidelines:

- Light color scheme to convey freshness and modernity. Primary color: a refreshing digital blue (#1F8EAF) for key elements and calls to action. Background color: a very subtle, desaturated cool grey-blue (#EAF6F8) providing a clean canvas. Accent color: a vibrant green (#2DE69B) to highlight urgent actions and special offers, symbolizing growth and natural elements.
- Headline font: 'Space Grotesk' (sans-serif) for a modern, slightly tech-forward feel. Body text font: 'Inter' (sans-serif) for excellent readability and a clean, neutral appearance across diverse product descriptions.
- Utilize a consistent set of clean, line-style icons. These should be easily recognizable and convey function simply, supporting navigation and interaction across all product categories.
- Implement a responsive, modular grid layout, optimized for displaying diverse product types efficiently. The design should prioritize intuitive navigation and visual clarity, adapting seamlessly to mobile and desktop views.
- Incorporate subtle, tasteful animations for transitions and feedback. This includes smooth product loading, cart additions, and hover effects, enhancing the overall user experience without distraction.
